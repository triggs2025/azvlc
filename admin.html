One-Time Setup

To save changes to the site, you need a GitHub token. You only have to do this once — it will be remembered on this device.

How to get a token:
1. Go to github.com/settings/tokens/new
2. Name it "AZVLC Admin"
3. Set expiration to No expiration
4. Check the repo scope
5. Click "Generate token" and paste it above

Managing Your AZVLC Site

Adding / Editing Policies

Click "+ Add New Policy" on the Policies tab. Fill out the form and hit Save. It saves directly to GitHub and the site updates automatically within a minute. Set "Approved" to Yes when you're ready for it to appear publicly.

Adding / Editing Politicians

Click "+ Add New Politician" on the Politicians tab. Fill out the form and hit Save. Grades represent the count of each rating received from Veterans.

Connecting GHL Webhooks

  1. In Go High Level, create a webhook under Automation → Workflows
  2. Go to your GitHub repo and edit js/app.js
  3. Find ghlWebhookPolicy and ghlWebhookRating near the top
  4. Paste your webhook URLs between the quotes and commit

Updating the Logo & Hero Image

Go to your GitHub repo, navigate to the assets/ folder, and upload logo.png and hero.jpg. The site updates automatically.

Your GitHub Token

The token is only stored in your browser's session. When you log out or close the tab, it's gone. You'll need to paste it again next time (or save it in a password manager).
